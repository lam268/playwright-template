# ttlab-playwright

## Getting started

This project is template for running automation test by using playwright and cucumber

Cucumber: https://cucumber.io/

Playwright: https://playwright.dev/

## Clone project

Running project with the following command:

```
yarn
# Generate report
yarn report
```
